## Image Specifications
- Format: JPG or PNG
- Recommended size: 200x200 pixels or larger (square aspect ratio preferred)
- File naming: Use lowercase with hyphens, matching the pattern in the code

## Adding Images
1. Place your officer photos in this `public/officers/` directory
2. Name them exactly as listed above
3. The images will automatically appear on the officers page

## Notes
- Images are optimized using Next.js Image component for better performance
- If an image is missing, the page will fall back to showing the officer's icon
- All officers now have image URLs configured - just add the photos to see them displayed
