## Image Specifications

- **Format**: JPG, PNG
- **Resolution**: Minimum 300x300 pixels (square aspect ratio recommended)
- **Size**: Under 2MB per image
- **Naming**: Use lowercase with hyphens (e.g., `first-last.jpg`)

## Notes

- Images will be displayed in 80x80 pixel containers with rounded corners
- If an image is missing, the page will gracefully fall back to displaying an icon
- Professional headshots are recommended for best visual impact
